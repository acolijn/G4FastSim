\doxysubsection{Detailed Description}
Initializes the actions for the simulation. 

This class is responsible for initializing the actions required for the simulation. It provides methods to build the actions for the master thread and for worker threads. The actions are built based on a \doxylink{classGammaRayHelper}{Gamma\+Ray\+Helper} object.
