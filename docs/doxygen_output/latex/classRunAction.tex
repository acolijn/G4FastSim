\doxysubsection{Detailed Description}
User-\/defined run action class for the simulation. 

This class inherits from G4\+User\+Run\+Action and is responsible for defining the actions to be taken at the beginning and end of each run. It also provides methods for initializing and defining various ntuples used for data storage. The class also includes setter methods for configuring the fast simulation, maximum number of scatters, maximum energy, and output file name.
