\doxysubsection{Detailed Description}
The \doxylink{classG4FastSim_1_1Hit}{Hit} class represents a hit in the detector. 

It inherits from the G4\+VHit class and contains properties such as energy deposit, position, time, track ID, parent ID, momentum, particle type, and process type.
